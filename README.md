# Flipkart Product Data Analysis (SQL Project)

## Overview
This project analyzes Flipkart product data using SQL to extract insights related to pricing, discounts, and brand performance.

## Objectives
- Understand product pricing trends
- Analyze discounts across categories
- Evaluate brand performance
- Explore product ratings

## Database Design
The dataset includes:
- Product details
- Pricing information
- Ratings and reviews
- Brand and category data

## Key SQL Operations Used
- SELECT, WHERE, GROUP BY
- Aggregate functions (COUNT, AVG)
- Window functions (RANK)
- Views

## Key Insights
- Top brands contribute a significant portion of total products
- Some categories offer discounts greater than 50 percent
- High-priced products above 10000 are relatively fewer
- Ratings vary across brands and categories

## Tools Used
- SQL (MySQL)

## Project Structure
- flipkart.sql (all queries and schema)
